# **drugz / WebSim Legacy Projects**

This repository hosts the **legacy WebSim projects** created back when WebSim was the platform where vibe used to code and experiment.  
All of those old projects now live here — fully open, editable, and ready for anyone to modify, extend, or remix.

These projects represent a snapshot of a creative era, preserved so others can learn from them, rebuild them, or transform them into something entirely new.

- **Archived WebSim code** from the original creative sessions  
- **Fully open-source projects** meant to be explored and modified

- **A playground** for developers who enjoy digging into raw, experimental code
